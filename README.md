# postcross

Depuis un terminal ou sur l'IDE.

Installer le projet :  
```
mvn clean install
```
Lancer le projet :  

```
mvn spring-boot:run
```

Ensuite il faut ouvrir le lien dans le navigateur web : http://localhost:8080/connect/facebook
